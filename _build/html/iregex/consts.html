Constants¶

Convenient constants for use in Regex.

iregex.consts.ALPHA = '[a-zA-Z]'¶

Any lower or uppercase character a-z A-Z

iregex.consts.ALPHA_NUMERIC = '\\w'¶

Any alphanumeric character a-z or 0-9

iregex.consts.ANY = '.'¶

Any character at all

iregex.consts.ASTERISK = '\\*'¶

Literal ASTERISK

iregex.consts.DOLLAR = '\\$'¶

Literal Dollar Sign

iregex.consts.END_OF_LINE = '$'¶

The end of a line

iregex.consts.LOWERCASE = '[a-z]'¶

Any lowercase character a-z

iregex.consts.NEWLINE = '(?:\\n|\\r\\n?)'¶

Indicates a newline, use in re.MULTILINE mode. Supports all operating systems

iregex.consts.NON_ALPHANUMERIC = '\\W'¶

Any non-alphanumeric character

iregex.consts.NON_NUMERIC = '\\D'¶

Any non-numeric character

iregex.consts.NON_WHITESPACE = '\\S'¶

Any non-whitespace character

iregex.consts.NUMERIC = '\\d'¶

Any numeric character 0-9

iregex.consts.ONE_OR_MORE = '+'¶

One or more of the last character or group

iregex.consts.OPTIONAL = '?'¶

Zero or one of the last character or group

iregex.consts.PERIOD = '\\.'¶

The literal period

iregex.consts.PLUS = '\\+'¶

Literal Plus

iregex.consts.QUESTION_MARK = '\\?'¶

Literal Question Mark

iregex.consts.START_OF_LINE = '^'¶

The start of a line

iregex.consts.UPPERCASE = '[A-Z]'¶

Any uppercase character A-Z

iregex.consts.WHITESPACE = '\\s'¶

Any whitespace character

iregex.consts.ZERO_OR_MORE = '*'¶

Zero or more of the last character or group
